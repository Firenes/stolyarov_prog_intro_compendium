Встроенные средства работы со строками

– length(string) – принимает в параметры строку.
  Возвращает целое число – длину строки.

– SetLength(string, integer) – изменяет длину строки.
  Принимает в себя строку для изменения и целое число для указания длины.

----------------------------
s := 'abrakadabra';
SetLength(s, 4); // -> 'abra'
----------------------------

Можно как укорачивать, так и удлинять строку. Во втором случае строка будет
забиваться мусором, если символов будет меньше, чем длины.

Но есть нюанс, если строка имеет фиксированное значение,
к примеру, s10: string[10], то при увеличении 10 ничего хорошего не будет.
Память уже выделена для 10 символов.

Поэтому функция SetLength() меняет длину строки, а не выделенную память.

– LowerCase(string) – принимает в параметры строку.
  Возвращает string в нижнем регистре.

– UpCase(string) – принимает в параметры строку.
  Возвращает string в верхнем регистре.

– copy(string, integer, integer) – принимает в параметры строку,
  начальную позицию, количество символов.
  Возвращает подстроку заданной строки начиная с заданной позиции
  с указанной длиной.

– delete(var string, integer, integer) – принимает в var-параметр строки,
  параметр начальной позиции, количество символов.
  Удаляет из строки начиная с заданной позиции с указанной длиной.

– insert(string, var string, integer) – принимает строку вставляемая строка,
  var-параметр строка, в которую вставляют,
  целое число – позииция, с которой вставляется строка.

– pos(string, string) – принимает строку для поиска, строка в которой ищут.
  Возвращает целое число – позиция, с которой начинается найденная
  подстрока.

– val(string, var longint/integer/byte, var word) – строит число типа
  longint, integer или byte по его строковому представлению.
  Принимает string – строка, в которой должна содержаться текстовое
  представление числа.
  longint/integer/byte – var-переменная, в которую передаётся число.
  word – var-переменная, в которую передаётся позиция в строке, где перевод
  дал сбой или 0, если перевод произошёл успешно.

– str(12.5:9:3, string) – псевдопроцедура по переводу числа в строку.
  12.5 – число, которое перерводится в строку
  9 – итоговое количество символов в строке
  3 – количество пробелов в начале.

– BinStr(longint/integer/byte, integer) – возвращает строку со строковым
  представлением числа в двоичной системе.
  Принимает в себя целое число любого типа для перевода в двоичную систему.
  Целое число для количества символов в строке.

– OctStr(longint/integer/byte, integer) – возвращает строку со строковым
  представлением числа в восьмеричной системе.
  Принимает в себя целое число любого типа для перевода в восьмеричную
  систему.
  Целое число для количества символов в строке.

– HexStr(longint/integer/byte, integer) – возвращает строку со строковым
  представлением числа в шестнадцатеричную системе.
  Принимает в себя целое число любого типа для перевода в шестнадцатеричную
  систему.
  Целое число для количества символов в строке.
