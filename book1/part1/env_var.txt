Программы в ОС UNIX могут получать инфу из переменнух окружения.
Переменные окружения –– множесто текстовых строк вида VAR=VALUE,
где VAR –– имя перменной, а VALUE –– её значение

Процесс приложения может добавлять, удалять или изменять свои значения.
Если процесс запускает другой процесс, то второй наследует окружение от родителя

PATH –– список каталогов, в которых следует искать исполняемый файл, если пользователь дал команду, не указав каталог
(выз $PATH)

HOME –– путь к домашнему каталогу пользователя
(выз $HOME)

LANG –– язык, на который ориентируются многоязычные приложения
(выз $LANG)

EDITOR –– предпочитаемый редактор текстов
(выз $LANG)

Просмотреть перменные окружения можно через вызов команды export

Переменные можно убрать из окружения с помощью команд unset и export
unset MYVAR
export MYVAR

Копирование значения переменных обратно в окружение с помощью команды export:
PATH=$PATH:/sbin:/usr/sbin
export PATH

Чтобы сменить инфорацию о пользователе, включая используемый командный интерпретатор, можно с помощью команды chfn, которая может быть реализована по-разному: в одних системах задаёт пользователю ряд вопросов, а в других –– предлагает к редактированию определэнный текст, из которого затем извлеает нужные значения.
